This package provides Salesforce RESTful API Http Middleware leveraging [Relay](http://relayphp.com/).

## Installation

To install, use composer:

```
composer require flipboxdigital/relay-salesforce
```

## Testing

``` bash
$ ./vendor/bin/phpunit
```
